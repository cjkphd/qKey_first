Data Types

  • Integer = 1, 10, 100, 150
  • Strings = A string of characters and letters, must be surrounded by " " or ' '
  • Boolean = True or False statement
  • Array = Contains multiple values inside brackets [ ] and separated by commas [1, A]
  • NAN - Not A Number

JavaScript

  • = in JavaScript assigns a Variable with a Value
  • == in JavaScript if both sides are the same. If it is, convert to a true, if not convert to false. Example (10 == 10 ) same as (true)
  • + - * / Are symbols used to calculate between integers
  • (MODULO) % takes the remainder of a division of two integers and can only be used for Integers
  • When you "Call" a variable or function, you want it to "Return" the value it's holding

If Statement

  • if (a boolean to run or not to run) {
    the code that runs if the boolean is true
    }

While Statement

  • while (use a boolean to run or not to run for each loop) {
    the code to run if the boolean is true
    }
WARNING if the boolean never turns FALSE, the while loop will run an infinite loop and crash the program.

For Statements

  • opt.(A)

    }
  • opt.(B)
    for (var = i; i < Some #; i++) {
    code here
    }

Comparitors

  • == Compare the left and right side, if they are equal, return a true, if not return a false
  • <= Compare the left and right side, if the left side is SMALLER or equal, return a true, otherwise return a false
  • >= Compares the left and right side, if the left side is GREATER or equal, return a true, otherwise return a false
  • != Compare left and right side, if they DO NOT equal, return a true, otherwise return a false
  • < Return a true if left side is SMALLER than right side
  • > Return a true if left side is BIGGER than the right side
